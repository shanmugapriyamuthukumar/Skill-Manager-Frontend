📘 Application Overview

Skill Manager is an internal admin system designed to manage employee skills, review and approve skill submissions, and align project skill requirements efficiently. It helps administrators maintain accurate skill data and ensure proper project resource allocation.

⚙️ System Capabilities

  • Project Skill Mapping
  • Employee Skill Tracking
  • Role‑Based Access Control
